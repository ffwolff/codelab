# CodeLab
### 🇧🇷 Português

## Sobre o Projeto

CodeLab é uma aplicação web interativa criada para demonstrar conceitos fundamentais de programação através de exemplos práticos e visualmente acessíveis.

Cada funcionalidade apresenta:

- Uma explicação do conceito ou algoritmo;
- Uma demonstração interativa executada no navegador;
- A implementação da lógica em JavaScript;
- Exemplos equivalentes em Python;
- Exemplos equivalentes em Java.

O objetivo é reunir conceitos básicos e intermediários de programação em uma plataforma única, servindo tanto como ferramenta de estudo quanto como demonstração de habilidades de desenvolvimento Front-End e lógica de programação.

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla JS)

### Validações

CodeLab is an interactive web application designed to demonstrate fundamental programming concepts through practical and easy-to-understand examples.

Each feature includes:

- An explanation of the concept or algorithm;
- An interactive demonstration running in the browser;
- The implementation in JavaScript;
- Equivalent examples in Python;
- Equivalent examples in Java.

The goal is to gather basic and intermediate programming concepts in a single platform, serving both as a learning resource and as a showcase of Front-End and programming skills.
